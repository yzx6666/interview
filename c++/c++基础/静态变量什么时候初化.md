# **静态变量什么时候初始化**

## 静态变量分为全局静态变量、局部静态变量、类静态变量。

<br/>

## 初始化方法分为静态初始化、动态初始化
## **静态初始化**
**指的是用常量来对静态变量初始化，包括0初始化和常量表达式初始化，其中0初始化的变量保存在`.bss段`。常量表达式初始化的变量保存在`.data段`。`对于静态初始化的变量，是在程序加载完成时初始化`**

<br/>

## **动态初始化**
**指的是需要调用函数才能完成的初始化，比如类的构造函数。对于全局或者类的静态成员变量，是在`main()`函数执行前由运行时调用相应的代码进行初始化的。而对于局部静态变量，是在函数执行至初始化语句时才开始执行初始化的。**
