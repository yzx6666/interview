# **内联函数和函数的区别，内联函数的作用**
1.  **内联函数比普通函数多了关键字```inline```**
2.  **内联函数避免了函数调用的开销，普通函数有调用的开销**
3.  **普通函数调用时，需要寻址（函数入口地址），内联函数不需要寻址**
4.  **内联函数有一定限制，要求代码`简单`,不能包含复杂的结构控制语句（例如 `for()`）** 

<br/>

## **内联函数和宏函数的区别**
1.  **宏定义不是函数，但是使用起来像函数，预处理器用复制宏代码的方式代替调用函数，节省了函数压栈退栈的过程，提高了效率**<br/>**内联函数本质是一个函数，内联函数用于函数体简单的函数，并且内联函数不能直接调用自己**
2.  **宏函数是在`预编译`的时候把所有的宏名用宏体代替，是字符串替换。<br/>内联函数是在`编译期`进行代码插入，编译器会在调用内联函数的地方直接展开内联函数的内容，省去了函数调用的开销**
3.  **宏函数没有类型检查，无论对错直接替换<br/>内联函数在编译的时候进行类型检查，内联函数满足函数的性质，如：参数列表、返回值等**

## **使用注意事项**
1.  **宏定义没有类型检查**
2.  **内联函数用于较小的、频繁调用的函数。使用只需在函数前加`inline`**
3.  **内联函数最好定义在头文件里，而不是只有声明：`因为编译器处理内联函数时要展开函数，所以只有声明是不够的`**
