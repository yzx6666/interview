# **静态局部变量、局部变量、全局变量的特点**

### 从作用域考虑：c++作用域可分为，全局、局部、类、语句、命名空间、文件作用域
1.  **全局变量：全局作用域，可以通过`extern`作用于其他非定义的源文件**
2.  **静态全局变量：全局作用域和文件作用域，所以无法在其他文件中使用**
3.  **局部变量：局部作用域，比如函数的参数、函数内的局部变量**
4.  **静态局部变量：局部作用域，只被初始化一次，直到程序结束`在局部变量前加static`**

### 从空间考虑:除了局部变量在栈上外，其他都在静态存储区。因为静态变量都在静态存储区，所以一次调用的时候还是取到原来的值

### 生命周期：局部变量在栈上，出了作用域就回收内存；全局变量、静态全局变量、静态局部变量都在静态存储区，直到程序结束才回收内存

`static 全局变量: 改变作用范围，不改变存储位置`(本来就在静态存储区)

`static 局部变量：改变存储位置，不改变作用范围`(改到静态存储区存储)
